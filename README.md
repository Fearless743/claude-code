本项目不仅仅是一个简单的反编译产物，它承载了以下核心价值：

## 快速开始 (Getting Started)

### 环境依赖

*   **Runtime**: [Bun](https://bun.sh/) >= 1.3.11 (必须使用最新版以规避 FFI 相关问题)
*   **System Tools**: 建议安装 `ripgrep` (rg) 以获得最佳搜索性能。

### 安装与部署

```bash
# 安装依赖
bun install

# 开发模式启动 (看到版本号 888 即代表链路通畅)
bun run dev

# 生产级构建
bun run build
```

构建产物位于 `dist/` 目录，支持在 Bun 或 Node.js 环境下独立运行。
